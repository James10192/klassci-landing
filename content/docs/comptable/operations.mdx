Ce guide couvre les **opérations courantes** du comptable une fois la configuration initiale des frais réalisée. Pour le **paramétrage initial** des catégories et des tarifs par classe, voir [Module Frais & Comptabilité](/docs/modules/frais-comptabilite).

## En préparation

Cette page est en cours de rédaction. Les sections prévues couvrent le quotidien de l'équipe comptable :

- **Encaisser un paiement** — enregistrer un règlement (espèces, virement, mobile money), via le caissier ou directement, générer le reçu.
- **Suivre les soldes** — par étudiant, par classe, par catégorie de frais. Lecture du dashboard comptable.
- **Configurer les relances automatiques** — seuils de retard, templates de messages (SMS / email), fréquence d'envoi.
- **Suivi par catégorie** — vue agrégée des recouvrements par type de frais, taux d'impayés.
- **Exports comptables** — formats Sage, Excel, CSV. Périmètres temporels (mois, trimestre, année).
- **Ajustements et remises** — appliquer une remise exceptionnelle (bourse interne, geste commercial), traiter un trop-perçu, gérer un avoir.
- **Clôture de fin d'année** — basculer les soldes restants sur l'année suivante, archiver les comptes des étudiants partis.
